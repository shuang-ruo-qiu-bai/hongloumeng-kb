## 核心信息

- **服务器**：`ssh root@114.67.247.66`，代码在 `/opt/hongloumeng/`
- **重启**：`systemctl restart hongloumeng`
- **技术栈**：Flask + Jinja2 + gunicorn + SQLite + Chroma RAG
- **所有代码**：`服务端代码/` 文件夹里有完整副本，可以直接改
- **80+ 维护脚本**：`临时脚本/` 按功能分类（sidebar/books/canon/poems/sunwen/search/misc）

## 当前最高优先级

1. **图书下载 404** — `library.html` 链接到 `/books/xxx` 但无 Flask 路由，添加 `send_from_directory`
2. **原著注释编号跳柱** — 正文中 `[5]` 和注释列表 `[1]` 对不上，API 重编号了注释但没替换正文
3. **AI 问答按钮不响应** — 疑似 CSS 偏移导致
4. **底部名言区消失** — `overflow-x: hidden` 可能影响滚动条

## 关于孙温绘本

服务器上有 104 张孙温《红楼梦》图画，存放在 `/opt/hongloumeng/app/static/sunwen/`。相关脚本在 `临时脚本/sunwen/` 下（18 个），涉及下载、横幅、导航、封面等各种处理。

## 工作方式

- 代码修改后需要 `scp` 到服务器并 `systemctl restart hongloumeng`
- 临时脚本模式：本地 `临时脚本/` 里写好 python 脚本，scp 到服务器执行
- 每次修改后递增 `?v=2` 的版本号（现在用的是 `v=2`）
