# 红楼梦寻 · 项目档案

> 最后更新：2026-06-08

## 项目定位

专为《红楼梦》读者、研究者和爱好者打造的红学知识平台。和通用 AI 聊天工具不同，只做红学。收录内容经过精选整理，由专业背景的编者辑录。

## 访问方式

| 途径 | 地址 | 说明 |
|------|------|------|
| 主站 | https://stoneseek.top | 通过 Cloudflare Tunnel 加速 |
| 推广页 | https://stoneseek.top/promo | 朋友圈分享专用 |
| 服务器 IP | http://114.67.247.66:5000 | 临时备用 |

域名 `stoneseek.top` 托管于腾讯云，DNS 由 Cloudflare 解析，通过 Cloudflare Tunnel 绕过国内服务器的 ICP 备案限制。

## 内容收录

- 《红楼梦》原著全文（一百二十回，脂评本系统）
- 孙温绘本红楼梦图（全套 203 幅）
- 红学文献、研究论文、学术资料
- 诗词大全、版本源流、风俗考据
- 红学历史与各派学者

后续将持续丰富板块，深化学术内容，跟进 AI 能力优化。

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | Python 3 + Flask + gunicorn（gthread, 1 worker, 4 threads） |
| 前端 | Jinja2 模板 + 原生 CSS/JS（无框架） |
| 认证 | Flask-Login |
| AI | LLM API（DeepSeek/OpenAI 等，可配置） |
| 搜索 | Chroma 向量库 + 句嵌入 ONNX 模型（768 维） |
| 数据库 | SQLite（用户/会话/消息） |
| 部署 | systemd + Cloudflare Tunnel（cloudflared） |

## 使用方式

1. **本地知识搜索**（免费，无需登录）：输入关键词，从知识库检索原文和文献
2. **AI 学术问答**（需登录）：基于本地知识库的 RAG + LLM，学术风格回答，提供文献出处

搜索支持三种模式：混合搜索（推荐）、关键词搜索、语义搜索。

## 部署架构

```
用户 → stoneseek.top (Cloudflare DNS)
       → Cloudflare Tunnel (cloudflared)
       → localhost:5000 (gunicorn + Flask)
```

Cloudflare Tunnel 建立从服务器到 Cloudflare 的出站连接，绕过国内服务器入站 Host 头检测，无需 ICP 备案即可使用国内云服务器 + 自定义域名。
