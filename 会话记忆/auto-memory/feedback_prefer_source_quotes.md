需要叙事效果时，优先引用源文件中的当事人原话、现场描写、情感表述，而不是自己推演。

**Why:** 用户指出，回忆录（黄金海、徐景贤）和通史（杨继绳）本身就有大量生动的原文——当事人的原话、对天气/场景的描写、心理活动的表述。直接引用这些原文就能达到故事效果，比我推演的"脸上带着青紫的伤痕"之类更有说服力、更严谨。

**How to apply:** 
1. 需要故事性细节时，先查源文件有没有相关原文可以引用
2. 当事人自己的叙述永远比推演更生动
3. 只有在原文完全没有相关叙述时，才做有限推演，且必须标注（推演）
4. 推演是最后手段，不是叙事工具
