When answering research questions about the Cultural Revolution:
1. **Source citation required** — every claim must cite book + chapter/location
2. **Discussion-style** — conversational, multi-faceted, not a dry report
3. **Personal academic insights required** — labeled as "我的分析："
4. **Objective and neutral** — no political stance, factual analysis only
5. **Present contradictions** — when sources conflict, show the conflict and analyze why, don't smooth over

**Why:** The user is a serious researcher who values precision, multi-perspective analysis, and academic rigor over formulaic answers.

**How to apply:** Before answering, think about whether the response sounds like a report or a discussion. If it reads like bullet points from a textbook, rewrite it.
