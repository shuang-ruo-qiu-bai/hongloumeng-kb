The user's knowledge base contains exclusively non-official, de-ideologized sources (港版《中华人民共和国史》, memoirs, independent scholarship). They do not use official PRC publications or state media as sources. Official narratives on these topics are vague and omit details, which is why the user deliberately avoids them.

When citing sources in answers, NEVER label any book in the corpus as "官方史书" or "official". The corpus is curated to contain only materials that differ from or are independent of official PRC narratives.

**Why:** The user actively curates their library to exclude propaganda and official positions.

**How to apply:** Always distinguish source types (港版学术著作 vs 回忆录 vs 独立研究), but never call any of them "官方". If a source IS official (rare in this corpus), label it explicitly and note the user may want to cross-verify against non-official sources.
