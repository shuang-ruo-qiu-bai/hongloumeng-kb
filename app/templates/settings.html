选择 AI 提供商

不同选项的优缺点：

DeepSeek

性价比高 · 中文优秀
需 DeepSeek API Key

Claude

中文最佳
需 Anthropic API Key

OpenAI

兼容好
需 OpenAI API Key

Ollama

完全免费 · 本地
需安装 Ollama

DeepSeek 去 platform.deepseek.com 获取；Ollama 不需要 Key
Claude 默认为 api.anthropic.com，OpenAI 默认为 api.openai.com/v1
DeepSeek 默认 deepseek-v4-flash，Claude 默认 claude-sonnet-4，OpenAI 默认 gpt-4o-mini
启用 AI 问答

获取 API Key

选择你需要的提供商并获取 Key：

DeepSeek（推荐）：在 platform.deepseek.com 注册 → API Keys → 创建 Key，充值后可用。价格极低，约 1 元可问数百次。

Claude：在 console.anthropic.com 获取 API Key

OpenAI：在 platform.openai.com 获取 API Key

Ollama（免费）：从 ollama.com 下载安装，运行 ollama pull qwen2.5:7b
