## 部署架构

```
用户 → stoneseek.top (Cloudflare DNS)
       → Cloudflare Tunnel (cloudflared)
       → localhost:5000 (gunicorn + Flask)
```

Cloudflare Tunnel 建立从服务器到 Cloudflare 的出站连接，绕过国内服务器入站 Host 头检测，无需 ICP 备案即可使用自定义域名。
